This project starts from zero.

Build a polished interactive educational web model of the Patek Philippe Grandmaster Chime.

Do not reuse any previous prototype files.
Do not copy Patek Philippe artwork, diagrams, photos, or proprietary movement layouts.
Use original simplified geometry.
Do not claim exact CAD accuracy.
The model should be mechanically plausible and educational, not a replica.

Project direction:
Use React, Vite, Three.js, @react-three/fiber, and @react-three/drei unless there is a strong reason not to.
Use a clean dark technical interface.
Keep the app easy to run locally.

Persistence rules:
Save work frequently.
Commit after every stable milestone.
Update HANDOFF.md after every major milestone.
If stopped, rate limited, blocked, or unsure, save files, commit the current working state, and update HANDOFF.md before ending.

Validation before committing:
Run npm install if needed.
Run npm run build.
Run npm run dev if possible.
Check that the app loads.
Check that there are no obvious console errors.
Check that controls work.
Check that the layout works on desktop and narrow screens.

Before stopping, HANDOFF.md must include:
Current status
Files changed
What works
What is broken
Next exact task
How to resume
