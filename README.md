# Grandmaster Chime 3D Educational Model

A from-scratch interactive educational web model inspired by the architecture of the Patek Philippe Grandmaster Chime.

> This is **not** an exact CAD replica. Geometry and placements are original, simplified, and mechanically plausible for learning.

## Stack
- React + Vite
- Three.js
- @react-three/fiber
- @react-three/drei

## Features
- Dark technical UI
- Interactive 3D watch assembly
- Orbit rotate/zoom controls
- Exploded movement toggle
- Front/calendar side switching
- Clickable mechanism groups
- Beginner/advanced explanation modes
- Complication panel (20 items)
- Energy flow visualization for strike path
- Chiming mode simulation:
  - Grande sonnerie
  - Petite sonnerie
  - Minute repeater
  - Date repeater
  - Alarm strike
- Source notes panel with abstraction disclaimer
- Responsive layout for desktop and narrow screens

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
